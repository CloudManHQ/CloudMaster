# Data Engineer 面试题实例答案

## Q1: 如何保证数据一致性与质量？
**答**：建立数据校验规则与质量指标，采用分层校验（入湖、加工、产出），并结合血缘与回溯机制定位问题。对关键指标设置报警与回滚。

## Q2: 批流一体如何落地？
**答**：统一数据模型与口径，采用同一套元数据与指标系统，实时与离线共享公共逻辑，使用流式补偿保证最终一致性。

## Q3: 上游数据格式变更导致故障怎么办？
**答**：在入口设置 schema 校验与兼容策略，发现变更先隔离再回滚，并与上游建立版本协商与变更公告机制。
