# 03 深度学习基础 - 小白版

> **一句话秒懂**: 深度学习就像教电脑"自己学习认东西"——不用人告诉它怎么识别猫,它通过看大量图片,像婴儿一样慢慢学会区分猫和狗!

## 这一章讲什么?

深度学习是让电脑变聪明的魔法!这一章会教你:

- **神经网络是怎么工作的** - 就像人脑里密密麻麻的神经元连接
- **怎么训练神经网络** - 让它从"笨蛋"变"学霸"的秘诀
- **为什么深度学习这么厉害** - 能认图片、听懂人话、下围棋

## 学习路线图

想象你在学骑自行车:

```
第一步:了解自行车结构 (神经网络核心)
   ↓
   你需要知道车轮、车把、刹车怎么用
   → 对应:神经元、激活函数、反向传播
   
第二步:练习平衡和转弯 (训练优化)
   ↓
   摔倒了爬起来,慢慢调整方向和速度
   → 对应:优化器、学习率、防止过拟合
```

## 关键术语速查

- **神经网络**: 像人脑一样的计算结构,由很多"神经元"连接而成
- **激活函数**: 决定神经元是否"激活"的开关
- **反向传播**: 找出错误原因,从后往前改正的过程
- **优化器**: 帮助神经网络学习的"教练"
- **学习率**: 学习的快慢速度
- **过拟合**: 死记硬背训练数据,不会举一反三
- **BatchNorm**: 让训练更稳定的技巧
- **Dropout**: 防止过拟合的方法(随机"遗忘"一些连接)
