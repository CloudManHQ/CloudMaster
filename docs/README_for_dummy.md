# AI Guru 知识库 - 新手导航

> **一句话概括**: 这是一份让你从零开始理解人工智能的"导游手册"，不需要任何数学和编程基础。

欢迎来到 **AI Guru** 知识库的小白版！这里会用最通俗的语言，帮你搞懂"人工智能到底是什么"以及"它是怎么工作的"。

## 学习路线图

不管你是谁，都可以按下面的顺序学习：

```
第1站：基础理论（了解 AI 背后的"常识"）
   ↓
第2站：机器学习（让电脑从数据中学东西）
   ↓
第3站：深度学习（模仿人脑的学习方式）
   ↓
第4站：选你感兴趣的方向 ↓
   ├── 自然语言处理（让电脑读懂人话，比如 ChatGPT）
   ├── 计算机视觉（让电脑看懂图片和视频）
   └── 强化学习（让 AI 像打游戏一样自己学策略）
   ↓
第5站：AI 工程化（把 AI 变成真正能用的产品）
   ↓
第6站：伦理与安全（让 AI 安全、公平地为人类服务）
```

## 学习小贴士

- **不用着急**：每次读一小节就够了，理解比速度重要
- **多联想**：把 AI 概念和你日常生活中的事情联系起来
- **不懂就跳**：遇到看不懂的内容可以先跳过，回头再看
- **动手试试**：有些章节有简单的代码示例，试着运行一下会更有感觉
