# 01 基础理论 (Fundamentals)

本章节涵盖了人工智能与机器学习最底层的科学支柱。

## 1.1 数学基础 (Mathematical Foundations)

### [线性代数 (Linear Algebra)](./Linear_Algebra/Linear_Algebra.md)
- **核心内容**: 张量、线性变换、特征值分解 (EVD)、奇异值分解 (SVD)。
- **AI 关联**: 模型参数表示、降维算法 (PCA)、注意力机制计算。

### [概率论与数理统计 (Probability & Statistics)](./Probability_Statistics/Probability_Statistics.md)
- **核心内容**: 贝叶斯定理、高斯分布、估计理论 (MLE/MAP)、信息论 (熵、KL 散度)。
- **AI 关联**: 处理不确定性、损失函数设计、变分推断。

## 1.2 计算机科学基础 (Computer Science Basics)

### [数据结构与算法 (Data Structures & Algorithms)](./Data_Structures_Algorithms/Data_Structures_Algorithms.md)
- **核心内容**: 计算图、拓扑排序、空间索引 (HNSW)、动态规划。
- **AI 关联**: 自动微分实现、高效向量检索、模型解码策略。

### [分布式系统 (Distributed Systems)](./Distributed_Systems/Distributed_Systems.md)
- **核心内容**: 通信原语 (All-Reduce)、并行策略 (Data/Model/Pipeline Parallelism)、ZeRO 优化。
- **AI 关联**: 大规模集群训练、千亿级参数模型扩展。
