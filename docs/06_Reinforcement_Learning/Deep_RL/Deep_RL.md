# 深度强化学习 (Deep RL)

深度神经网络与强化学习的结合，开启了解决高维状态空间问题的可能。

## 1. 代表性算法 (Key Algorithms)

### DQN (Deep Q-Network)
- **创新**: 引入经验回放 (Experience Replay) 和 目标网络 (Target Network)。
- **来源**: [Playing Atari with Deep Reinforcement Learning (Mnih et al., 2013)](https://arxiv.org/abs/1312.5602)

### PPO (Proximal Policy Optimization)
- **特点**: 通过裁剪策略更新步长，平衡训练稳定性与效率。LLM 对齐的核心算法。
- **来源**: [Proximal Policy Optimization Algorithms (OpenAI, 2017)](https://arxiv.org/abs/1707.06347)

## 2. 进阶领域
- **离线强化学习 (Offline RL)**: 从离线数据集中学习，无需在线交互。
- **多智能体强化学习 (MARL)**: 处理多个智能体的博弈与协作。
